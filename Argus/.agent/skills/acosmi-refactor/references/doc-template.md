# 文档模板集 — Argus-Compound

> 版本：v1.1 · 2026-02-16

本文件包含改造过程中各类文档的标准模板。所有文档存放于 `docs/` 目录下。

---

## 一、任务跟踪模板

**路径**：`docs/renwu/task-YYYYMMDD-<topic>.md`

**命名规范**：以 `task-` 开头，区别于 `audit-`（审计报告）和 `bootstrap-`（上下文文档）。

```markdown
# {任务名称} 任务跟踪

> 创建日期：YYYY-MM-DD
> 状态：待启动 | 执行中 | 已完成
> 来源：[相关审计或需求文档链接]

## 背景与目标

简述为什么做、做什么、期望结果。

## 任务清单

### Batch A: {批次名}

| 状态 | 编号 | 任务 | 关联发现 |
|------|------|------|----------|
| ⬜ | A1 | ... | ... |
| ⬜ | A2 | ... | ... |

> **状态标记**：⬜ 待做 → 🔄 进行中 → ✅ 完成 → ⏭️ 跳过/延迟

## 完成后需更新的文档

- [ ] 来源文档（审计报告等）— 标记已修复项
- [ ] bootstrap 文档 — 标记已完成阶段
- [ ] deferred-items.md — 登记跳过/延迟项（如有）

## 风险与注意事项

- ...

## 变更记录

| 日期 | 变更内容 |
|------|----------|
```
