## 4. 本轮修复要点

- 飞书 `SendMessage` 对齐结构化错误模型：
  - sender 不可用 -> `SendErrUnavailable`
  - 文本发送失败 -> `SendErrUpstream`（`operation=send.text`）
  - 媒体失败且无文本兜底 -> `SendErrUpstream`（`operation=send.media`）
  - 空请求 -> `SendErrInvalidRequest`（`operation=send.validate`）
- 修复潜在风险：媒体发送失败时，若无文本不再“空成功”。
