## 分析

### cli/ + commands/（6% — 但有结构原因）

TS 的 cli/ 和 commands/ 合计 49,672L，包含：

- Commander.js 命令注册 + 选项解析
- 174 个独立命令文件（每个 80-400L）
- TUI 界面（Ink/React 渲染）
- 交互式向导

**Go 差异原因**：

1. Cobra 框架比 Commander.js 更紧凑（~60% 代码量）
2. 很多命令功能仅调用 Gateway WS 方法（已在 gateway/ 实现）
3. TUI 部分（src/tui/ 4,155L）是客户端专属，Go 不需要
4. 真正缺失的是**命令注册和路由逻辑**

### 高覆盖模块

- **plugins/hooks/cron/daemon/acp** 均 71-170% 覆盖
- 这些模块已在 Phase 6/11/12 审计中确认完整

## 关键缺失

| 缺失内容 | 估算行数 | 说明 |
|----------|----------|------|
| CLI 命令注册 | ~5,000 | 174 个命令 → Cobra subcommands |
| TUI 渲染 | ~4,155 | 客户端专属，Go 可能不需要 |
| cli-runner 集成 | ~500 | 命令执行管线 |

## Phase 6 评估
