# `openacosmi pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
openacosmi pairing list whatsapp
openacosmi pairing approve whatsapp <code> --notify
```
