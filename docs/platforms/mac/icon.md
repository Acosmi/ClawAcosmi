# macOS 应用图标

## 图标规格

- 格式：`.icns`（macOS 应用图标集）
- 包含所有标准大小：16, 32, 64, 128, 256, 512, 1024 像素
- 位置：`apps/macos/OpenAcosmi/Assets.xcassets/AppIcon.appiconset/`

## 菜单栏图标

- 模板图标（单色），支持自动暗色/亮色模式适配
- 大小：18x18pt（@2x 为 36x36px）
- 不同状态的动画变体（空闲/活跃/语音/错误）
