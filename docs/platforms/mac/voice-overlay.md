# 语音覆盖层（macOS 应用）

## 概览

语音覆盖层是 Talk 模式和语音唤醒期间显示的浮窗 UI，提供视觉反馈。

## 状态和动画

- **监听**：云朵随麦克风音量脉动
- **思考**：下沉动画（等待模型回复）
- **朗读**：辐射光环（TTS 播放中）

## 交互

- 点击云朵：停止当前朗读
- 点击 X：退出 Talk 模式
- 支持拖拽重新定位

## Talk 模式流程

1. 语音唤醒词触发 → 显示覆盖层
2. 语音转写（Speech 框架）→ 显示部分文本
3. 静默检测 → 发送给 Go Gateway（`chat.send`）
4. 等待模型回复 → 思考动画
5. TTS 播放回复（ElevenLabs 流式）→ 朗读动画
6. 播放完成 → 返回监听状态

## Push-to-Talk

- 按住右 Option 键立即开始捕获（无需唤醒词）。
- 覆盖层在按住期间显示部分文本。
- 释放后短暂延迟然后发送。
- Push-to-Talk 期间暂停唤醒词检测以避免冲突。

## 设置

- **语音唤醒** 开关：启用唤醒词运行时
- **按住 Cmd+Fn 说话** ：启用 Push-to-Talk（macOS 26+）
- 语言和麦克风选择器
- 触发词表
- 音效：触发检测和发送时的提示音
