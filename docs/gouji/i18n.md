| 属性 | 值 |

后端 CLI/TUI 国际化模块，支持双语言（中文 zh-CN / 英文 en-US）。覆盖 onboarding 向导、频道配置、技能管理、浏览器控制、Tailscale 连接等用户交互界面。

## 核心接口

```go
func T(key string, params map[string]string) string  // {key} 参数替换
func Tp(key string) string                            // 无参数翻译
func Tf(key string, args ...interface{}) string       // fmt.Sprintf 风格
func InitFromEnv()                                     // 环境变量自动检测
func RegisterBundle(lang Lang, messages map[string]string)  // 扩展语言包
```
