# 微信公众号内容创作

本技能为 oa-media 子智能体提供微信公众号图文内容创作的完整知识体系。

## 平台限制

| 项目 | 限制 |
|------|------|
| 标题 | ≤64 字符 |
| 正文 | ≤50000 字 |
| 封面图 | 900×500px，JPG/PNG |
| 正文配图 | 宽 640px，≤200KB |
| 图片总数 | ≤500 张/篇 |
| 发布后修改 | 标题改 3 字，正文改 20 字，图片改 3 张 |

## 爆款标题公式

标题决定打开率。控制在 **25 字以内**，最佳 15-20 字。

### 核心公式

1. **数字 + 痛点**：「3 个方法让你的代码效率提升 50%」
2. **好奇悬念**：「为什么 90% 的程序员都不知道这个技巧？」
3. **对比反差**：「月薪 5000 和月薪 50000 的人，差距只在这一点」
4. **热点借势**：「{热点话题} + 你最该知道的 3 件事」
5. **清单型**：「2026 年必须掌握的 10 个 AI 工具」
6. **痛点 + 解决方案**：「总加班？这套工作流帮你每天省 2 小时」

### 标题禁忌

- 不做标题党（算法会降权）
- 不用纯英文标题
- 不在标题中放链接

## 正文结构

### 基本框架

```
开头（前 3 行）：直击痛点或抛出问题 — 决定是否继续阅读
├── 小标题 1：核心论点 + 案例/数据
├── 小标题 2：深入分析 + 实操方法
├── 小标题 3：延伸思考 + 对比
结尾：总结要点 + 引导互动（点赞/在看/转发）
```

### 写作要点

- 每段 3-5 行，避免长段
- 重点内容**加粗**标注
- 使用有序/无序列表增强可读性
- 引用数据和案例增加可信度
- 结尾设置互动话题，引导留言

## 排版规范

### 字体与间距

| 元素 | 推荐值 |
|------|--------|
| 正文字号 | 15px |
| 一级标题 | 18-20px |
| 二级标题 | 16-17px |
| 行间距 | 1.75 倍 |
| 段间距 | 空一行 |
| 字间距 | 1px |
| 页边距 | 10px |

### 配色

- 正文：深灰 `#3f3f3f`（避免纯黑）
- 强调色：最多 2 种（品牌主色 + 辅色）
- 背景：白色为主

### 图片规范

- 封面图：900×500px，16:9 比例
- 正文配图：宽 640px，统一比例
- 文件大小：≤200KB
- 每张图下方添加说明文字（14px 灰色居中）
- 使用无版权图源：Unsplash、Pexels

## 发布检查清单

发布前逐项确认：

- [ ] 标题 ≤64 字符，简洁有力
- [ ] 封面图清晰，尺寸 900×500
- [ ] 正文无错别字，格式统一
- [ ] 配图清晰、风格统一
- [ ] 超链接可正常跳转
- [ ] 已设置原创声明（如适用）
- [ ] 已预览移动端显示效果

## 使用 oa-media 工具的流程

```
1. trending_topics → 获取热点话题
2. web_search → 补充素材和数据
3. 用 LLM 能力按上述公式撰写标题和正文
4. image → 生成封面图和配图
5. content_compose(draft) → 保存草稿
6. 回报主智能体 → 等待审批
7. media_publish(publish) → 审批通过后发布
```
